# Death In Poverty Clipper

Audio clipper VST3 plugin built with JUCE for true Soundcloud rappers.

## Features

- Hard and Soft clipping modes
- Input/Output Gain control
- Adjustable threshold
- Dry/Wet mix
- Modern GUI with custom rotary sliders

## Building

### Requirements

- JUCE 7.0+
- Visual Studio 2022 (Windows)
- Xcode (macOS)
- CMake 3.15+

### Windows

1. Open `DeathInPovertyClipper.jucer` in Projucer
2. Click "Save and Open in IDE"
3. Build in Visual Studio

### macOS

1. Open `DeathInPovertyClipper.jucer` in Projucer
2. Click "Save and Open in IDE"
3. Build in Xcode

## Installation

Copy the built plugin to your VST3 folder:

- **Windows**: `C:\Program Files\Common Files\VST3\`
- **macOS**: `~/Library/Audio/Plug-Ins/VST3/`
